# Sentiment Analysis NLP Project

## Overview
Machine Learning model to classify movie reviews into Positive or Negative.

## Dataset
IMDb Movie Reviews (50,000 reviews)

## Model
- TF-IDF Vectorizer
- Naive Bayes Classifier

## Accuracy
~86%

## Example Predictions
- Positive review → 1  
- Negative review → 0  

## Tools
Python, Scikit-learn, NLP
